# Flask Weather Application ☁️

A Flask-based Weather Forecast Application deployed using Docker, Kubernetes, and Jenkins CI/CD pipeline.

---

# 🚀 Tech Stack

* Flask
* Docker
* Jenkins
* Kubernetes (K3s)
* GitHub
* Docker Hub

---

# ⚙️ Features

* Live Weather API
* Dockerized Flask App
* Jenkins CI/CD Pipeline
* Kubernetes Deployment
* Automatic Docker Image Build
* Automatic Kubernetes Deployment

---

# 🔄 CI/CD Workflow

```text
Git Push
   ↓
Jenkins Pipeline
   ↓
Docker Build
   ↓
Docker Hub Push
   ↓
Kubernetes Deployment
```
